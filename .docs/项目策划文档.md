# 贪吃蛇游戏 - 项目策划文档

---

## 1. 项目概述

### 1.1 项目名称
MyAITestProject - 贪吃蛇游戏

### 1.2 项目简介
本项目是使用Unreal Engine 5开发的经典贪吃蛇游戏的现代化重制版，在经典玩法基础上增加了多种食物类型、障碍物系统和存档功能。

### 1.3 项目定位
- **游戏类型**：休闲益智类
- **目标用户**：全年龄段
- **游戏平台**：PC（可扩展至其他平台）

### 1.4 开发周期
- 当前状态：已完成核心功能开发

## 2. 游戏玩法设计

### 2.1 核心玩法

#### 2.1.1 基本规则
- 玩家控制蛇在网格地图上移动
- 蛇头触碰边界时**从对侧穿出**（环绕机制，不触发游戏结束）
- 碰到自身时游戏结束（隐身状态下可穿越自身）
- 碰到障碍物时游戏结束（无敌状态摧毁障碍物；隐身状态穿越障碍物）
- 吃到食物后蛇身变长，分数增加
- 每次吃到食物后会生成一个新障碍物

#### 2.1.2 操作方式

| 操作 | 按键 | 说明 |
|------|------|------|
| 向上移动 | W / 上箭头 | 控制蛇向上移动 |
| 向下移动 | S / 下箭头 | 控制蛇向下移动 |
| 向左移动 | A / 左箭头 | 控制蛇向左移动 |
| 向右移动 | D / 右箭头 | 控制蛇向右移动 |
| 加速 | 长按任意方向键 | 蛇移动速度加快 |

### 2.2 游戏元素

#### 2.2.1 蛇（Snake）
- 由蛇头 + 蛇身段组成
- 蛇头：玩家控制的主体
- 蛇身段：跟随蛇头移动，吃到食物后增加

#### 2.2.2 食物系统

| 食物类型 | 出现概率 | 视觉表现 | 效果 |
|----------|----------|----------|------|
| 普通食物 | 70% | 基础外观 | 蛇身变长 +1，分数 +10 |
| 隐身食物 | 10% | 半透明/特殊颜色 | 蛇身变长 +1，分数 +10，10秒内不检测自身碰撞 |
| 无敌食物 | 20% | 发光/特殊颜色 | 蛇身变长 +1，分数 +10，10秒内可摧毁障碍物 |

#### 2.2.3 障碍物系统
- 每次吃到食物后随机生成一个障碍物
- 障碍物占据一个网格位置
- 蛇碰到障碍物时游戏结束
- 无敌状态下蛇可以摧毁障碍物
- 游戏开始时清空所有障碍物

### 2.3 游戏流程

#### 2.3.1 启动阶段
1. 游戏启动，加载主菜单
2. 显示最高分列表
3. 等待玩家点击"开始游戏"

#### 2.3.2 游戏阶段
1. 隐藏主菜单，显示游戏UI
2. 生成蛇和初始食物
3. 玩家控制蛇移动
4. 吃到食物 → 蛇身变长 → 分数增加 → 生成新食物和障碍物
5. 触发食物特效（如适用）

#### 2.3.3 结束阶段
1. 触发游戏结束条件
2. 保存当前分数到最高分
3. 显示游戏结束界面
4. 显示本次分数和最高分排名
5. 等待玩家重新开始

### 2.4 分数系统
- 每个食物：+10分
- 最高分保存Top 10记录
- 自动排序显示

## 3. 技术架构

### 3.1 技术栈

| 技术 | 版本/说明 |
|------|-----------|
| 游戏引擎 | Unreal Engine 5.3+ |
| 编程语言 | C++ |
| UI框架 | UMG (Unreal Motion Graphics) |
| 可视化 | 蓝图 |

### 3.2 核心类架构

#### 3.2.1 游戏模式层
- **SnakeGameMode**：游戏入口，设置默认PlayerController和HUD

#### 3.2.2 游戏管理层
- **SnakeManager**：核心管理器，负责：
  - 游戏状态管理（分数、游戏结束）
  - 生成/销毁游戏对象
  - 相机管理
  - 边界可视化
  - 存档管理

#### 3.2.3 游戏对象层
- **Snake**：蛇类，继承APawn
  - 移动控制
  - 蛇身管理
  - 碰撞检测
  - 特效管理
- **SnakeSegment**：蛇身段
- **Food**：食物类
  - 多种类型
  - 收集动画
- **SnakeObstacle**：障碍物类

#### 3.2.4 输入控制层
- **SnakePlayerController**：玩家控制器
  - 输入处理
  - 加速逻辑

#### 3.2.5 UI层
- **SnakeHUD**：HUD管理
- **SnakeMenuUI**：菜单UI
- **SnakeGameUI**：游戏UI

#### 3.2.6 数据层
- **SnakeSaveGame**：存档类
