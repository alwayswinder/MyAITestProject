# 贪吃蛇游戏项目总体说明

## 项目概述

这是一个使用Unreal Engine 5开发的经典贪吃蛇游戏，使用C++实现核心逻辑，支持蓝图扩展。

## 技术栈

- Unreal Engine 5.3+
- C++
- UMG（Unreal Motion Graphics）
- 蓝图

## 游戏玩法

### 基本规则
- 控制蛇在网格地图上移动
- 吃到食物后蛇身变长，分数增加
- 碰到边界或自身时游戏结束

### 操作方法

#### 移动控制
- **W 键 / 上箭头**：向上移动
- **S 键 / 下箭头**：向下移动
- **A 键 / 左箭头**：向左移动
- **D 键 / 右箭头**：向右移动

#### 加速机制
- 长按任意方向键即可加速
- 松开后恢复正常速度

## 游戏流程

1. **游戏启动**
   - SnakeGameMode设置默认PlayerController和HUD
   - HUD创建SnakeMenuUI并显示
   - 显示鼠标光标

2. **开始游戏**
   - 玩家点击"开始游戏"按钮
   - SnakeMenuUI隐藏，鼠标隐藏
   - SnakeManager生成Snake和Food
   - SnakeGameUI显示
   - 相机切换到SnakeManager的相机

3. **游戏进行中**
   - 玩家通过方向键控制蛇的移动
   - 长按方向键可以加速
   - 蛇吃到食物后分数增加，蛇身变长
   - Food在新的随机位置重新生成

4. **游戏结束**
   - 蛇碰到边界或自身时触发GameOver
   - Snake停止移动
   - SnakeGameUI隐藏
   - SnakeMenuUI重新显示（标记为GameOver状态）
   - 旧的Snake和Food被销毁

## 输入设置

在Unreal Engine编辑器的项目设置中需要配置以下输入：

- **MoveUp**: 绑定到 W 键和 上箭头键
- **MoveDown**: 绑定到 S 键和 下箭头键
- **MoveLeft**: 绑定到 A 键和 左箭头键
- **MoveRight**: 绑定到 D 键和 右箭头键

## 蓝图使用

### 创建蓝图类

1. **WBP_SnakeMenuUI**:
   - 父类: `SnakeMenuUI`
   - 包含: 开始游戏按钮（命名为 `StartGameButton`）

2. **WBP_SnakeGameUI**:
   - 父类: `SnakeGameUI`
   - 包含: 分数显示文本
   - 可以调用 `Get Score` 函数获取当前分数

### 在关卡中设置

1. 放置 `SnakeManager` Actor到关卡中
2. 配置SnakeManager的属性：
   - 设置 `Snake Class` 为Snake蓝图
   - 设置 `Food Class` 为Food蓝图
   - 调整 `Boundary Distance X/Y` 来设置游戏区域大小
   - 调整 `Grid Size` 来设置网格大小
3. 在World Settings中设置Game Mode为 `SnakeGameMode`
4. 在SnakeHUD蓝图或Details面板中设置：
   - `Snake Menu UI Class` 为 `WBP_SnakeMenuUI`
   - `Snake Game UI Class` 为 `WBP_SnakeGameUI`

## 注意事项

1. 确保所有蓝图类都正确设置了父类
2. 确保UI中的控件名称与代码中的 `BindWidget` 匹配
3. Snake的HeadMesh和SnakeSegment的Mesh需要在编辑器中设置
4. Food的Mesh需要在编辑器中设置
5. BoundaryMesh需要在编辑器中设置一个平面网格

## 扩展功能建议

1. **添加音效**:
   - 吃到食物的音效
   - 游戏结束的音效
   - 移动的音效

2. **添加粒子效果**:
   - 吃到食物的粒子特效
   - 游戏结束的粒子特效

3. **添加难度递增**:
   - 随着分数增加，移动速度逐渐加快
   - 增加障碍物

4. **添加存档系统**:
   - 保存最高分
   - 保存游戏进度

5. **添加多人模式**:
   - 本地多人对战
   - 网络多人对战
