# 贪吃蛇游戏项目 - 完整需求说明

## 一、项目概述

**项目名称：** MyAITestProject（贪吃蛇游戏）
**开发引擎：** Unreal Engine 5.3+
**主要语言：** C++ + 蓝图
**项目类型：** 休闲益智类游戏

请帮我从零开始创建一个完整的贪吃蛇游戏项目，使用C++实现核心逻辑，配合蓝图进行UI和资源配置。

## 二、技术栈要求

- **游戏引擎**：Unreal Engine 5.3 或更高版本
- **编程语言**：C++（核心逻辑）+ 蓝图（UI和配置）
- **UI框架**：UMG (Unreal Motion Graphics)
- **构建配置**：确保 Build.cs 包含以下模块：Core, CoreUObject, Engine, InputCore, UMG, Slate, SlateCore

## 三、核心功能需求

### 3.1 基本游戏规则

1. **移动控制**：
   - W/上箭头：向上移动
   - S/下箭头：向下移动
   - A/左箭头：向左移动
   - D/右箭头：向右移动
   - 长按方向键0.5秒后触发加速

2. **边界处理**：
   - 蛇头触碰边界时**从对侧穿出**（环绕机制），不触发游戏结束

3. **碰撞检测**：
   - 碰到自身时游戏结束（隐身状态下可穿越自身）
   - 碰到障碍物时游戏结束（无敌状态可摧毁障碍物；隐身状态可穿越障碍物）

4. **食物系统**：
   - 普通食物（70%概率）：蛇身变长+1，分数+10
   - 隐身食物（10%概率）：蛇身变长+1，分数+10，10秒内不检测自身碰撞
   - 无敌食物（20%概率）：蛇身变长+1，分数+10，10秒内可摧毁障碍物

5. **障碍物系统**：
   - 每次吃到食物后随机生成一个障碍物
   - 障碍物占据一个网格位置

### 3.2 存档系统

- 自动保存最高分，最多保存10条记录
- 分数按从高到低排序
- 存档槽名称：SnakeHighScores

## 四、核心C++类实现（按顺序）

请按以下顺序创建C++类，确保依赖关系正确：

### 第一阶段：数据和基础类

1. **SnakeSaveGame** - 存档类，继承USaveGame
   - 保存最高分列表（Top 10）

2. **SnakeSegment** - 蛇身段类，继承AActor
   - 简单的静态网格组件

3. **SnakeObstacle** - 障碍物类，继承AActor
   - 静态网格组件，无特殊逻辑

### 第二阶段：游戏对象类

4. **Food** - 食物类，继承AActor
   - 定义EFoodType枚举：Normal, Invisible, Invincible
   - 70%普通、10%隐身、20%无敌
   - 收集动画：弹跳上升+缩放消失，持续0.5秒
   - 委托OnFoodTypeChanged
   - 碰撞检测：吃到后触发分数增加、蛇身变长、特效
   - 每次收集后生成障碍物

5. **Snake** - 蛇类，继承APawn
   - 蛇头+蛇身段管理
   - 边界环绕机制
   - 隐身效果：跳过自身碰撞和障碍物碰撞
   - 无敌效果：碰到障碍物时摧毁它
   - 加速机制：长按方向键0.5秒后加速
   - 委托：OnInvisibleStarted/Ended、OnInvincibleStarted/Ended

### 第三阶段：控制和UI类

6. **SnakePlayerController** - 玩家控制器，继承APlayerController
   - 输入处理：绑定MoveUp/MoveDown/MoveLeft/MoveRight
   - 加速逻辑：长按检测

7. **SnakeGameUI** - 游戏UI，继承UUserWidget
   - 显示当前分数

8. **SnakeMenuUI** - 菜单UI，继承UUserWidget
   - 开始游戏按钮（命名为StartGameButton）
   - 显示最高分列表
   - 游戏结束时显示本次分数

9. **SnakeHUD** - HUD类，继承AHUD
   - 管理SnakeMenuUI和SnakeGameUI的创建、显示、隐藏

### 第四阶段：管理器和游戏模式

10. **SnakeManager** - 游戏管理器，继承AActor
    - 包含相机组件（俯视-90度）
    - 边界平面可视化
    - 随机有效位置生成（避免与蛇、食物、障碍物重叠）
    - 存档管理（Top 10最高分）
    - StartGame()：重置分数、清除障碍物、生成蛇和食物、显示游戏UI
    - GameOver()：保存最高分、停止蛇、隐藏游戏UI、显示菜单UI

11. **SnakeGameMode** - 游戏模式，继承AGameModeBase
    - 设置默认PlayerController为SnakePlayerController
    - 设置默认HUD为SnakeHUD

## 七、关键实现注意事项（非常重要）

1. **边界碰撞**：蛇头超出边界时**环绕到对侧，不触发游戏结束
2. **隐身效果**：跳过自身碰撞和障碍物碰撞
3. **无敌效果**：碰到障碍物时摧毁它，自身碰撞仍然触发游戏结束
4. **加速机制**：长按方向键0.5秒后触发加速
5. **分数系统**：每个食物+10分，保存Top 10最高分
6. **障碍物生成**：每次吃到食物后生成一个障碍物
