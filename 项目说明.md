# 贪吃蛇游戏项目说明文档

## 项目概述

这是一个使用Unreal Engine 5开发的经典贪吃蛇游戏，使用C++实现核心逻辑，支持蓝图扩展。

## 核心类说明

### 1. SnakeGameMode（游戏模式）
**文件**: `SnakeGameMode.h` / `SnakeGameMode.cpp`

- 继承自 `AGameModeBase`
- 负责设置默认的PlayerController和HUD
- 是游戏的入口点

### 2. SnakeManager（游戏管理器）
**文件**: `SnakeManager.h` / `SnakeManager.cpp`

**主要功能**:
- 管理游戏状态（分数、游戏结束等）
- 生成和销毁Snake和Food
- 提供边界可视化（BoundaryMesh）
- 提供游戏相机（CameraComponent + SpringArmComponent）
- 管理UI的显示和隐藏

**可配置属性**:
- `Score`: 当前分数
- `FoodScoreValue`: 每次吃到食物增加的分数
- `GridSize`: 网格大小
- `BoundaryDistanceX` / `BoundaryDistanceY`: 边界距离
- `SnakeClass`: Snake类
- `FoodClass`: Food类

**主要函数**:
- `StartGame()`: 开始游戏
- `GameOver()`: 游戏结束
- `UpdateBoundaryMesh()`: 更新边界网格
- `UpdateCameraDistance()`: 更新相机距离

### 3. Snake（蛇）
**文件**: `Snake.h` / `Snake.cpp`

**主要功能**:
- 控制蛇的移动和方向
- 管理蛇身段
- 检测碰撞（边界和自身）
- 处理加速逻辑

**可配置属性**:
- `MoveSpeed`: 正常移动速度
- `BoostMoveSpeed`: 加速移动速度
- `InitialSegmentCount`: 初始蛇身段数量
- `SnakeSegmentClass`: 蛇身段类

**主要函数**:
- `StartGame()`: 初始化蛇
- `ChangeDirection()`: 改变移动方向
- `EatFood()`: 吃到食物，增加蛇身
- `GameOver()`: 游戏结束
- `StartBoost()`: 开始加速
- `StopBoost()`: 停止加速

### 4. SnakeSegment（蛇身段）
**文件**: `SnakeSegment.h` / `SnakeSegment.cpp`

- 简单的Actor类，代表蛇的身体段
- 包含StaticMeshComponent用于可视化

### 5. Food（食物）
**文件**: `Food.h` / `Food.cpp`

**主要功能**:
- 在随机位置生成食物
- 检测与蛇的碰撞
- 碰撞后重新生成

**主要函数**:
- `SpawnAtRandomLocation()`: 在随机位置生成食物
- `OnBeginOverlap()`: 碰撞检测回调

### 6. SnakePlayerController（玩家控制器）
**文件**: `SnakePlayerController.h` / `SnakePlayerController.cpp`

**主要功能**:
- 处理玩家输入
- 控制蛇的移动方向
- 处理长按加速逻辑

**输入绑定**:
- `MoveUp`: 向上移动（W/上箭头）
- `MoveDown`: 向下移动（S/下箭头）
- `MoveLeft`: 向左移动（A/左箭头）
- `MoveRight`: 向右移动（D/右箭头）

**加速机制**:
- 长按任意方向键即可加速
- 松开后恢复正常速度

### 7. SnakeHUD（HUD）
**文件**: `SnakeHUD.h` / `SnakeHUD.cpp`

**主要功能**:
- 创建和管理UI Widget
- 提供获取UI的接口

**可配置属性**:
- `SnakeMenuUIClass`: 菜单UI类
- `SnakeGameUIClass`: 游戏UI类

**主要函数**:
- `GetSnakeMenuUI()`: 获取菜单UI
- `GetSnakeGameUI()`: 获取游戏UI

### 8. SnakeMenuUI（菜单UI）
**文件**: `SnakeMenuUI.h` / `SnakeMenuUI.cpp`

**主要功能**:
- 显示开始游戏菜单
- 显示游戏结束界面
- 处理按钮点击事件

**主要函数**:
- `ShowUI()`: 显示UI
- `OnStartGameClicked()`: 开始游戏按钮回调

### 9. SnakeGameUI（游戏UI）
**文件**: `SnakeGameUI.h` / `SnakeGameUI.cpp`

**主要功能**:
- 显示游戏过程中的信息（如分数）
- 提供获取分数的接口

**主要函数**:
- `ShowUI()`: 显示UI
- `HideUI()`: 隐藏UI
- `GetScore()`: 获取当前分数

## 游戏流程

1. **游戏启动**:
   - SnakeGameMode设置默认PlayerController和HUD
   - HUD创建SnakeMenuUI并显示
   - 显示鼠标光标

2. **开始游戏**:
   - 玩家点击"开始游戏"按钮
   - SnakeMenuUI隐藏，鼠标隐藏
   - SnakeManager生成Snake和Food
   - SnakeGameUI显示
   - 相机切换到SnakeManager的相机

3. **游戏进行中**:
   - 玩家通过方向键控制蛇的移动
   - 长按方向键可以加速
   - 蛇吃到食物后分数增加，蛇身变长
   - Food在新的随机位置重新生成

4. **游戏结束**:
   - 蛇碰到边界或自身时触发GameOver
   - Snake停止移动
   - SnakeGameUI隐藏
   - SnakeMenuUI重新显示（标记为GameOver状态）
   - 旧的Snake和Food被销毁

## 输入设置

在Unreal Engine编辑器的项目设置中需要配置以下输入：

- **MoveUp**: 绑定到 W 键和 上箭头键
- **MoveDown**: 绑定到 S 键和 下箭头键
- **MoveLeft**: 绑定到 A 键和 左箭头键
- **MoveRight**: 绑定到 D 键和 右箭头键

## 蓝图使用

### 创建蓝图类

1. **WBP_SnakeMenuUI**:
   - 父类: `SnakeMenuUI`
   - 包含: 开始游戏按钮（命名为 `StartGameButton`）

2. **WBP_SnakeGameUI**:
   - 父类: `SnakeGameUI`
   - 包含: 分数显示文本
   - 可以调用 `Get Score` 函数获取当前分数

### 在关卡中设置

1. 放置 `SnakeManager` Actor到关卡中
2. 配置SnakeManager的属性：
   - 设置 `Snake Class` 为Snake蓝图
   - 设置 `Food Class` 为Food蓝图
   - 调整 `Boundary Distance X/Y` 来设置游戏区域大小
   - 调整 `Grid Size` 来设置网格大小
3. 在World Settings中设置Game Mode为 `SnakeGameMode`
4. 在SnakeHUD蓝图或Details面板中设置：
   - `Snake Menu UI Class` 为 `WBP_SnakeMenuUI`
   - `Snake Game UI Class` 为 `WBP_SnakeGameUI`

## 扩展功能建议

1. **添加音效**:
   - 吃到食物的音效
   - 游戏结束的音效
   - 移动的音效

2. **添加粒子效果**:
   - 吃到食物的粒子特效
   - 游戏结束的粒子特效

3. **添加难度递增**:
   - 随着分数增加，移动速度逐渐加快
   - 增加障碍物

4. **添加存档系统**:
   - 保存最高分
   - 保存游戏进度

5. **添加多人模式**:
   - 本地多人对战
   - 网络多人对战

## 注意事项

1. 确保所有蓝图类都正确设置了父类
2. 确保UI中的控件名称与代码中的 `BindWidget` 匹配
3. Snake的HeadMesh和SnakeSegment的Mesh需要在编辑器中设置
4. Food的Mesh需要在编辑器中设置
5. BoundaryMesh需要在编辑器中设置一个平面网格
